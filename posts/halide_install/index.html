Installing Halide in Ubuntu/Pop OS with vcpkg

January 15, 2022 2-minute read

Instructions from Halide blogs

  1. Install cmake if your don’t have it already.

Download the precompiled binaries from: https://cmake.org/download/

If you prefer installing from the source, follow the instructions on the page.

  1. The second step is to install halide with vcpkg. Run the following command once you have installed vcpkg.
xxx@xxx-os:~$ ./vcpkg/vcpkg install halide:x64-linux

After some log ..

Total elapsed time: 55.61 min

The package halide provides CMake targets:

set(CMAKE_CXX_STANDARD 17)
set(CMAKE_CXX_STANDARD_REQUIRED YES)
set(CMAKE_CXX_EXTENSIONS NO)

find_package(Halide REQUIRED)

# JIT mode:
add_executable(my_halide_app main.cpp)
target_link_libraries(my_halide_app PRIVATE Halide::Halide)

# AOT mode:
add_executable(my_generators my_generators.cpp)
target_link_libraries(my_generators PRIVATE Halide::Generator)
add_halide_library(my_first_generator FROM my_generators)
add_halide_library(my_second_generator FROM my_generators
    PARAMS parallel=false scale=3.0 rotation=ccw output.type=uint16)
add_halide_library(my_second_generator_2 FROM my_generators
    GENERATOR my_second_generator
    PARAMS scale=9.0 rotation=ccw output.type=float32)
add_halide_library(my_second_generator_3 FROM my_generators
    GENERATOR my_second_generator
    PARAMS parallel=false output.type=float64)

For more information see: https://github.com/halide/Halide/blob/v13.0.2/README_cmake.md `

This appears to have installed Halide. To confirm installation, let’s try the first tutorial from the halide github page -

lesson_01_basics.cpp

To do this - I created a file with the same name in my test. I installed vcpkg in my home directory. So all my halide files are stored in the vcpkg folder in my home directory.

To compile the file - I ran:

g++ lesson_01*.cpp -g -I ~/vcpkg/installed/x64-linux/include -L ~/vcpkg/installed/x64-linux/lib -lHalide -lpthread -ldl -o lesson_01 -std=c++17
LD_LIBRARY_PATH=~/vcpkg/installed/x64-linux/lib ./lesson_01

And once the program compiled, the following “Success” string was returned by the program.

Success!

Great, if you have reached here, it meant you have successfully installed Halide and run a sample program calling into Halide libraries!
